# Flutter

## Setup
- [flutter `getting-started`](https://flutter.io/docs/get-started/install)
- VS Code
    - Dart Extension
    - Flutter Extension

## Step-by-step

### Check `flutter dev environment`
```
$ flutter doctor
```

- VS Code
```
Flutter: Run flutter doctor
```

### Create `foo` sample project
```
$ flutter create foo
```

- VS Code
```
Flutter: New Project
```

### Run project
```
$ flutter run
```

- VS Code
```
Flutter: Hot refresh
Flutter: Hot reload
```
